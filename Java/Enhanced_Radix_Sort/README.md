## Synopsis

Enhanced Radix Sort Algorithmn, switching to Insertion Sort when you have a small group of elements left to sort.
After some tests I discovered that in the most cases the best time to change from Radix Sort to Intertion sort is when you have 10 elements left to sort.

## Motivation

Course: CSE 2050 - Algorithmns and Data Structures, Fall 2015, Florida Institute of Technology
